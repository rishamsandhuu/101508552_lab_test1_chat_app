# Real-Time Chat Application

A simple real-time chat application built using **Node.js, Express, Socket.io, and MongoDB**.  
This project was created for a lab test and demonstrates authentication, room-based chat, private messaging, and message persistence.

---

## Technologies Used
- Node.js
- Express.js
- Socket.io
- MongoDB (Mongoose)
- HTML, CSS, Bootstrap
- jQuery, fetch API

---

## Features
- User signup with unique username
- User login/logout (session stored in localStorage)
- Join and leave predefined chat rooms
- Real-time room-based chat
- Private 1-to-1 messaging
- Typing indicator for private chat
- Messages stored in MongoDB
